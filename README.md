# openriro

리로스쿨 스크래핑 wrapper 라이브러리입니다.

## 주의사항

> API가 수시로 변경될 수 있습니다.

> 스크래핑 특성상, 사이트 업데이트에 따라 라이브러리가 작동하지 않을 수 있습니다. 또한, 학교마다 다른 설정으로 인해 일부 학교에서는 제대로 작동하지 않을 수 있습니다. 적극적인 참여를 환영합니다.
